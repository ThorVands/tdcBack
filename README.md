# 🐾 DogCamp Backend

## 📝 Resumo
Uma API REST moderna desenvolvida para o ecossistema **DogCamp**. O objetivo principal é servir como ponte entre o banco de dados e as interfaces de usuário, gerenciando hospedagens, reservas e cadastros de pets com eficiência.

## 🚀 Tecnologias
* **Node.js** 🟢 - Ambiente de execução.
* **TypeScript** 🔵 - Tipagem estática para maior segurança.
* **Express** 🚂 - Framework rápido e minimalista para rotas.
* **MongoDB** 🍃 - Banco de dados NoSQL para armazenamento flexível.
* **CORS** 🔓 - Liberação de acesso para consumo do Frontend.
* **JSON** 🛠️ - Formato padrão para troca de dados.
